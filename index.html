My Proposal

from Initial design to deployment
I can help build your app

I follow an Agile Methodology for effective development workflow. On specific tools - I use Jira to keep track on the progress and Git for version control.

Front-End Development

I can create fully responsive and functional front-end applications with React.js and JavaScript. This portfolio site is solely created with HTML, CSS, Bootstrap and JavaScript.

Back-End Development

I sometimes pair my front-end code with regular and/or asynchronous Node.js and it's Express framework. I can implement third party API's (ex: I am using getform.io for the Start The Conversation section) and/or create my own. I continuously use Postman to test all of my back-end work throughout the process so the finished product turns out ok .

DevOps & Deployment

Before it's time to deploy a project to the internet, I think of the dreaded wall of confusion in the software delivery process - so because of this I trust the DevOps Toolchain. I use continuous version control and testing tools such as Git and recently React Testing Library. When working in a team, I am familiar with pre-deployment staging and continuous release approvals from the Scrum master. Prior to deployment, infrastructure configuration for clean code is a personal "must have".
ex: This portfolio is being deployed with Netlify.
